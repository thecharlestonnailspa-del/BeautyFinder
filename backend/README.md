# Backend

Backend services live here:

- `api`: NestJS + Prisma API
- `api-py`: FastAPI migration scaffold
- `worker`: background processing worker

Useful root commands:

```bash
npm run dev:backend:api
```

```bash
npm run dev:backend:worker
```

```bash
npm run start:backend:api
```

```bash
npm run dev:backend:api-py
```

Database schema and seed scripts are in `prisma/`.
