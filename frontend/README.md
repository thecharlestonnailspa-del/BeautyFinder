# Frontend

Frontend apps live here:

- `admin-panel`: Next.js admin web app
- `owner-dashboard`: Next.js owner web app
- `mobile-app`: Expo customer app

Useful root commands:

```bash
npm run dev:frontend:admin
```

```bash
npm run dev:frontend:owner
```

```bash
npm run dev:frontend:mobile
```

Shared frontend-facing types come from `packages/types`.
