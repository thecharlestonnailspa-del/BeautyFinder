# Customer Mobile App

Customer-facing mobile client for:

- discovering `Nail` and `Hair` businesses
- booking appointments
- messaging businesses
- leaving reviews
- managing favorites and notifications

Current local implementation:

- cute Hello Kitty-inspired home screen
- live reads from the mock API when available
- falls back to seeded sample data offline
- web export available via `npm run build --workspace @beauty-finder/mobile-app`
